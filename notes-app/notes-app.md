## notes API

### Note Structure

we have a title of the note, and a body of note

### Note Methods

- listNotes: Lists titles of all available notes
- removeNote: removes the note of which the title is specified
- readNote: prints the note of which the title is specified
- addNote: add note with the given title and body

Thats all we will require
